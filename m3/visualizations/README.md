# Visualizations

Charts are generated here when the analysis scripts are run.

| Script | Charts produced |
|--------|----------------|
| `project1_sales_analysis/analysis.py` | `monthly_sales.png`, `category_performance.png`, `regional_sales.png`, `customer_segments.png`, `top_products.png` |
| `project2_marketing_analysis/analysis.py` | `channel_roi.png`, `conversion_rates.png`, `budget_vs_revenue.png`, `campaign_performance.png` |
| `project3_operations_analysis/analysis.py` | `completion_rates.png`, `dept_duration.png`, `cost_by_dept.png`, `priority_analysis.png` |

Run any analysis script with:

```bash
python <script_path>
```

All PNG files are written to this directory automatically.
